# Bloom Shop Website 🌐

## 🔹 Project Description

Bloom Shop is a modern and fully responsive e-commerce website developed using HTML5, CSS3, and Bootstrap 5. This project showcases a professional online shopping interface where users can explore premium sneaker collections through an attractive and user-friendly design. The main goal of this project was to practice responsive web design, Bootstrap components, product card layouts, hover effects, and modern UI development.

## 🔹 What is Included in This Project

✅ Product Showcase Section
✅ Interactive Product Cards
✅ Hover Effects
✅ Quick View Button Overlay
✅ Wishlist (Heart Icon) Feature UI
✅ Responsive Grid Layout
✅ Professional Footer Section
✅ Mobile Friendly Design

## 🔹 How This Project is Made

The complete website structure is created using HTML5. Bootstrap 5 Grid System is used to create a fully responsive layout across different screen sizes. CSS3 is used for custom styling, card animations, hover effects, transitions, and modern UI design. Product cards include image zoom effects, quick-view overlays, and interactive elements to improve user experience.
